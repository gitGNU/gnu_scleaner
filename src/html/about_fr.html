Aide rapide


Présentation
scleaner vous permet de supprimer des fichiers qui ne sont plus nécessaires au bon fonctionnement de votre ordinateur.
scleaner utilise des greffons d'entrée pour trouver des fichiers inutiles.
Ensuite scleaner utilise des greffons de sortie pour traiter les fichiers sélectionnés par l'utilisateur parmis les fichiers trouvés par les greffons d'entrée.


Important
Il est important de comprendre que scleaner peut fonctionner dans 2 modes :
  • Le premier, en mode simple (Les fichiers seront recherchés dans /home/user). Ce mode est activé lorsque scleaner est lancé par un utilisateur normal.
  • En mode administrateur (Les fichiers seront recherchés dans /home/ et dans divers autres emplacements définis dans les greffons d'entrée nécessitant root). Ce mode est activé lorsque scleaner est lancé en tant que superutilisateur (ou sudo)


Utilisation
Premièrement cliquez sur la première icone de la toolbar pour sélectionner les greffons d'entrée que vous voulez utiliser.
Sur la seconde, pour lancer l'analyse du disque dur. Durand cette étape, scleaner recherche les fichiers.
Enfin la troisème icone, utilise le plugins de sortie sélectionné, pour traiter les fichiers.
