Quick help


Introduction
scleaner allow you to clean files which are not anymore need by your system.
scleaner used input plugins to find potential files.
Then scleaner used output plugins to process selected files from these return by input plugins.


Important
scleaner have two running modes:
  • Single user mode (Files are search in /home/user). This mode is activated when scleaner is launched as standard user.
  • Root user mode (Files are search in /home and in various places defined by all Root input plugins). This mode is activated when scleaner is launcher as root or using sudo.


Usage
First click on the first toolbar's icon to find files using input plugins.
The second icon will launch scan (scleaner will find files).
Finally the third icon will process selected input files using selected output plugin.
