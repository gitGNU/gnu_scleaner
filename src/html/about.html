Quick help


Scan: Launch a full scan of
  • /home/user for simple user mode
  • /home/ and various location in root mode

When a scan was made, you can see a NoteBook that allow you to select files from input plugins

On the up you see a RadioBox that allow you to select an output plugin

Important: If some files are disabled in Treeview, that mean that you haven't launch scleaner as root or using sudo. And so you haven't access rights to manipulate these files. The status can be consulted in statusbar (third box) at any moment.
